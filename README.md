# PromptIdeas: AI Image Gallery with Prompts

![Project Banner](https://img.shields.io/badge/GPT-Image%20Prompt-blueviolet?style=for-the-badge&logo=OpenAI)

This project is a collection of AI-generated images, each paired with a precise prompt used for image generation models (such as GPT Nano, Banana, and all major image engines). Hover over any image to see its prompt and easily copy it for your own use or inspiration.

## Features
- Gallery of AI-generated images
- See the exact prompt on image hover
- One-click prompt copy
- Great for inspiration and creative reuse

## Example Prompt Command
```
Generate an image using a precise and suitable prompt for GPT Nano, Banana, or any image generation engine.
```
